# Kata Machine in C

# Usage
## 1. Bootstrap the build system.
```sh
cc -o nob nob.c
```

## 2. Generate the exercises.
```sh
./nob generate
```

## 3. Complete an exercise.
Use your editor to complete the exercises in `exercises/day<n>/`.

## 4. Test your solution.
```sh
./nob test <exercise_name>
```
